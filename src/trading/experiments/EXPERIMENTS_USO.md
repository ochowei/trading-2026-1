## 失敗嘗試記錄 (Failed Experiments)

以下實驗經測試後未能超越 USO-001，已刪除程式碼，僅保留結果記錄供未來參考。

### USO-002: Pullback 7% + CPF ≥ 40%（2026-03-28 測試）

**假設**：收緊回檔門檻至 7% + 加入 Close Position Filter ≥ 40%，過濾仍在急跌的訊號。參考 GLD-007 成功經驗。

| 指標 | Part A | Part B |
|---|---|---|
| 訊號數 | 31 (6.2/年) | 11 (5.5/年) |
| 勝率 | 48.4% | 63.6% |
| 累計報酬 | **-13.35%** | +7.05% |
| MDD | -9.59% | -5.20% |

**失敗原因**：雙重收緊（回檔 + CPF）過度過濾 Part A 的好訊號。Part B 表現提升但 A/B 嚴重不平衡。

### USO-003: WR ≤ -85 + TP +3.5%（2026-03-28 測試）

**假設**：收緊 WR 至 -85 提高進場品質 + 提高 TP 至 +3.5% 捕捉更多均值回歸幅度，TP/SL 對稱。

| 指標 | Part A | Part B |
|---|---|---|
| 訊號數 | 45 (9.0/年) | 19 (9.5/年) |
| 勝率 | 55.6% | 47.4% |
| 累計報酬 | +13.51% | **-3.78%** |
| MDD | -10.93% | -6.31% |

**失敗原因**：Part A 大幅改善但 Part B 崩潰。TP +3.5% 在 2024-2025 較平靜的市場中太高，許多本可 +3.0% 達標的交易因無法觸及 +3.5% 而被停損。典型的 A/B 不平衡過擬合。

### USO-004: CPF ≥ 30% 輕量過濾（2026-03-28 測試）

**假設**：保持 USO-001 所有參數不變，僅加入輕量 Close Position Filter ≥ 30%，隔離 CPF 效果。

| 指標 | Part A | Part B |
|---|---|---|
| 訊號數 | 34 (6.8/年) | 16 (8.0/年) |
| 勝率 | 41.2% | 62.5% |
| 累計報酬 | **-27.35%** | +5.91% |
| MDD | -9.59% | -4.40% |

**失敗原因**：即使 30% 的輕量 CPF 也嚴重損害 Part A。USO 的超賣反彈訊號常出現在 close position 低的日子（收盤接近日低），CPF 反而過濾掉這些好訊號。**結論：Close Position Filter 不適用於 USO。**

### 跨實驗教訓

1. **USO-001 的 A/B 平衡（1.0:1）極為珍貴**：三次嘗試都破壞了這個平衡
2. **Close Position Filter 不可跨資產通用**：GLD 有效不代表 USO 有效，原油超賣反彈的微觀結構不同
3. **USO 的 contango + 高波動特性限制了策略改進空間**：+2.85%/+5.16% 可能已接近該資產均值回歸策略的天花板
