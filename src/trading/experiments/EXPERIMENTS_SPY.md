## SPY-001: 回檔 + Williams %R + 反轉K線 (Pullback + WR + Reversal Candle)

### 設計理念 (Design Rationale)

SPY 作為 S&P 500 指數 ETF，日波動率 ~1.0-1.2%，與 GLD 相近。
依據跨資產教訓 Section 12「新資產啟動流程」，選用 GLD-007 模板（低波動率 < 2%）。
參數幾乎不需縮放（vol ratio ≈ 1.0）。

### 進場條件 (Entry Conditions)

| # | 指標 | 門檻 | 說明 |
|---|------|------|------|
| 1 | Pullback | ≥ 3% from 10d high | 近期有顯著回檔 |
| 2 | Williams %R(10) | ≤ -80 | 短期超賣確認 |
| 3 | Close Position | ≥ 40% of day range | 日內反轉跡象（收盤從低點反彈） |
| 4 | Cooldown | 7 天 | 防止訊號密集觸發 |

### 出場參數 (Exit Parameters)

| 參數 | 值 | 說明 |
|------|-----|------|
| 止盈 (Profit Target) | +3.5% | 配合低波動資產 |
| 止損 (Stop Loss) | -4.0% | 風險控管 |
| 持倉天數 (Holding Days) | 20 天 | 給予足夠回歸時間 |
| 追蹤停損啟動 | +2.0% | 獲利達 2% 後啟動 |
| 追蹤停損距離 | 1.5% | 從最高點回撤 1.5% 出場 |

### 成交模型 (Execution Model)

| 項目 | 設定 |
|------|------|
| 進場模式 | `next_open_market`（隔日開盤市價買入） |
| 止盈出場 | `limit_order Day` |
| 止損出場 | `stop_market GTC` |
| 到期出場 | `next_open_market` |
| 滑價 | 0.1% |
| 悲觀認定 | 是（同日觸及止盈+止損時，認定為止損） |
| 追蹤停損 | 啟動 +2.0%，距離 1.5% |

### 回測結果 (Backtest Results)

| 指標 | Part A (IS) | Part B (OOS) | Part C (Live) |
|------|------------|-------------|--------------|
| 期間 | 2019-2023 | 2024-2025 | 2026-今 |
| 訊號數 | 20 | 4 | 2 |
| 成交數 | 20 | 4 | 2 |
| 成交率 | 100% | 100% | 100% |
| 勝率 | 70.0% | 75.0% | 50.0% |
| 平均報酬 | -0.02% | -0.33% | -1.23% |
| 累計報酬 | -1.27% | -1.41% | -2.50% |
| 最大回撤 | -5.06% | -4.07% | -3.23% |
| 每年訊號 | 4.0 | 2.0 | 8.7 |

**觀察：**
- 勝率不錯（A: 70%, B: 75%），但累計報酬為負
- 原因：追蹤停損過早出場，多數獲利交易僅 +0.5~1.8%，但止損交易 -4.1% 完全抵銷
- Part A 出場分佈：達標 3 / 停損 17（含追蹤停損 9）/ 到期 0
- Part A/B 訊號比：4.0/yr vs 2.0/yr = 2.0:1（邊界可接受）
- **下一步建議：** 調整追蹤停損參數（提高啟動門檻或加大距離），或嘗試無追蹤停損版本
